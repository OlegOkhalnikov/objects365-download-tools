# Objects365 Dataset Download Tools

Набор скриптов для загрузки, проверки и восстановления датасета [Objects365](https://www.objects365.org/) с китайских серверов.

## 📦 Состав инструментов

| Скрипт | Назначение |
|--------|------------|
| `download_objects365.sh` | Загрузка всех 51 архивов с автоматическим переключением методов и докачкой |
| `check_targz.sh` | Проверка целостности загруженных архивов |
| `redownload_bad.sh` | Выборочная перезагрузка поврежденных архивов |

## 🚀 Быстрый старт

```bash
# 1. Загрузить все архивы
./download_objects365.sh

# 2. Проверить целостность
./check_targz.sh

# 3. Если есть битые - перезагрузить
./redownload_bad.sh
```

## 📊 Характеристики датасета

- **Количество архивов**: 51 (`patch0.tar.gz` - `patch50.tar.gz`)
- **Общий размер**: 327 ГБ
- **После распаковки**: ~1.5 ТБ
- **Требуемое место**: минимум 2 ТБ

## ⚠️ Особенности

- Автоматическое переключение между HTTPS/HTTP при ошибках SSL
- Поддержка докачки прерванных файлов
- Проверка целостности каждого архива
- Подробное логирование всех операций

## 📄 Лицензия и использование

Данные скрипты предназначены для загрузки датасета Objects365, который доступен **только для академических целей**. Пожалуйста, ознакомьтесь с официальными условиями использования датасета на [сайте Objects365](https://www.objects365.org/download.html) перед загрузкой.
