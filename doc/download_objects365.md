# download_objects365.sh - Скрипт для загрузки датасета Objects365

## 📋 Описание
Скрипт для надежной загрузки 51 архива датасета Objects365 (`patch0.tar.gz` - `patch50.tar.gz`) с поддержкой докачки и автоматическим переключением между различными методами загрузки при возникновении ошибок.

## 🔧 Основные возможности
- **Умная докачка** - продолжение загрузки с места остановки
- **5 методов загрузки** с автоматическим переключением при ошибках
- **Проверка существующих файлов** перед началом загрузки
- **Агрессивные таймауты** для предотвращения зависаний
- **Подробное логирование** всех операций

## 📂 Параметры
- `DOWNLOAD_DIR="object365/images/train"` - директория для сохранения архивов
- `LOG_FILE="download_objects365.log"` - файл лога
- Диапазон архивов: `patch0.tar.gz` - `patch50.tar.gz` (51 файл)

## 🚀 Использование
```bash
chmod +x download_objects365.sh
./download_objects365.sh
```

## 🔄 Методы загрузки (переключение при ошибках SSL/таймаутов)
1. **HTTPS (агрессивный)** - aria2c с проверкой сертификатов
2. **HTTP (агрессивный)** - aria2c без SSL
3. **curl** - с поддержкой докачки
4. **wget** - классический загрузчик
5. **aria2c (минимальный)** - упрощенная версия с HTTP

## 📊 Процесс загрузки
1. **Проверка существующих файлов** (пропуск корректно загруженных >3GB)
2. **Загрузка по одному файлу** последовательно
3. **Автоматические повторные попытки** (до 10 раз)
4. **Сброс зависших процессов** при превышении таймаута (300с)

## ✨ Особенности
- **Принудительное завершение** зависших процессов (`pkill`)
- **Проверка размера** (>3GB для корректного файла)
- **Очистка временных файлов** aria2 при повреждении
- **Адаптивная пауза** между попытками
- **Детальный прогресс** загрузки

## 📈 Вывод информации
- ✅ Зеленый - успешная загрузка/существующий файл
- ⚠️ Желтый - предупреждения (поврежденные файлы)
- ❌ Красный - ошибки загрузки
- 📊 Статистика: загружено/осталось/успешно/с ошибками

## 📝 Пример лога
```

```

## ⚙️ Технические детали
- **Таймаут операции**: 300 секунд
- **Таймаут соединения**: 10-30 секунд
- **Макс. попыток**: 10
- **Пауза между попытками**: 5-10 секунд
- **Параллельные соединения**: 2-4 (aria2c)
- **Проверка размера**: >3GB (фактический размер ~4GB)

## 🛡️ Обработка ошибок
- SSL/сертификаты → переключение на HTTP
- Таймауты → принудительный сброс процесса
- Частичная загрузка → автоматическая докачка
- Зависание → `timeout` + `pkill`
