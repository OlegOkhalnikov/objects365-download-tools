# redownload_bad.sh - Скрипт для перезагрузки поврежденных архивов

Перед использованием рекомендуется запустить `check_targz.sh` для выявления битых архивов и обновить массив `BAD_ARCHIVES` в скрипте.

## 📋 Описание
Скрипт для выборочной перезагрузки поврежденных архивов датасета Objects365, выявленных при проверке целостности. Использует несколько методов загрузки с автоматическим переключением.

## 🎯 Назначение
Перезагрузка конкретных битых архивов из предопределенного списка с проверкой целостности после загрузки.

## 📂 Параметры
- `DOWNLOAD_DIR="object365/images/train"` - директория с архивами
- `LOG_FILE="redownload_bad.log"` - файл лога
- `BAD_ARCHIVES` - массив с номерами поврежденных архивов

## 🚀 Использование
```bash
chmod +x redownload_bad.sh
./redownload_bad.sh
```

## 🔄 Методы загрузки (последовательное переключение)
1. **aria2c** (HTTPS) - многопоточная загрузка с докачкой
2. **wget** - классический загрузчик с поддержкой докачки
3. **curl** - альтернативный метод с повторными попытками

## 📊 Процесс работы
1. **Удаление** старого битого файла и временных данных aria2
2. **Последовательная загрузка** каждым методом до успеха
3. **Проверка целостности** после каждой загрузки (`tar tzf`)
4. **Логирование** успешных и неудачных загрузок
5. **Пауза 5 секунд** между архивами

## ✨ Особенности
- **Целевая перезагрузка** только проблемных архивов
- **Автоматическое переключение** методов при ошибках
- **Проверка целостности** после каждой загрузки
- **Детальный вывод** размера загруженных файлов в GB
- **Итоговая статистика** успешных/неудачных загрузок

## 📈 Вывод информации
- ✅ Успешная загрузка с указанием метода и размера
- ⚠️ Предупреждения о сбоях методов
- ❌ Финальный список не загруженных архивов

## ⚙️ Технические детали
- **Таймаут**: 300 секунд
- **Попытки**: 3 для каждого метода
- **Параллельные соединения**: 8 (aria2c)
- **Пауза между архивами**: 5 секунд

## 📝 Пример лога
```
✅ Успешно: patch12.tar.gz
✅ Успешно: patch13.tar.gz
❌ Не удалось: patch14.tar.gz
```
